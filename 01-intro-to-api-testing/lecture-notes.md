# Application Programming Interface

> APIs are contractsthat enable communication:
> * web site talking to a data base
> * mobile application  talking to another application service

---

## The API Contract 

* Request                        
  * Endpoint
  * Headers
  * Body
    
*  Response
   * Status Code
   * Headers
   * Body
---
> APIs are public! For example [Spotify](https://developer.spotify.com/documentation/web-api)
---
